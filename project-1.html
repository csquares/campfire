Mobile Restaurant App

Description

This is a restaurant prototype mobile application. I wanted the user to have quick access to their favorites and have a count at it's finger tips. During checkout I decided to incorporate a dessert option to encourage last minute items. Once the user hits process payment I added an interaction button. This project is part of the Adobe daily challenge series, but I decided to take it further than what the challenge asks for.


Challenge

Design the mobile ordering version for a restaurant. Create components and reuse them in your project. Decide within twenty four hours what layout to use. What components require interaction.

Duration

  • 24 hrs

Role

  • UI Designer
  • Interaction Designer

Tools

  • Adobe XD

Inspiration

Layout Design

The landing section is broken down into favorites section, menu filter section, and items vertical scroll section. I felt that having a horizontal scrolling filter would work best to narrow the food items displayed below. Allowing each individual container to have its own item information would allow for future data to be added.

Components

The components re-used for this project were the favorites item, shopping bag, listing item, check out section, and the purchase button.
